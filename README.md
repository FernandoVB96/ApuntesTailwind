# Apuntes de Estilos con Tailwind CSS

## 1. Colores (Textos y Fondos)

### Textos
```html
text-{color}-{shade}
```
Ejemplo:
```html
text-red-500: Texto de color rojo con una intensidad de 500.


bg-{color}-{shade}

### Fondo
```html
bg-{color}-{shade}
```
Ejemplo:
```html
bg-green-300: Fondo verde con una intensidad de 300.
```
